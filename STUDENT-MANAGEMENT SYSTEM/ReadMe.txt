# Student Management System (Python Tkinter + SQLite)

A simple desktop **Student Management System** built using **Python Tkinter** and **SQLite database**.

## Features

* Add student records
* Store data using SQLite database
* View students in a professional table
* Prevent duplicate student IDs
* Simple and clean graphical interface

## Technologies Used

* Python
* Tkinter (GUI)
* SQLite3 (Database)

## How to Run

1. Clone the repository

```
git clone https:/joshuamakwale/github.com//student-management-system.git
```

2. Run the program

```
python main.py
```
