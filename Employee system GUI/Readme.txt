# Employee Management System (Python + Tkinter)

## Overview

This project is a desktop Employee Management System built using Python and Tkinter.
It allows users to add, update, delete, and search employee records through an interactive graphical interface.

The system stores employee data in a JSON database and displays records in a dynamic table.

This project demonstrates practical Python development skills including GUI design, data management, validation, and object-oriented programming.

---

## Features
*log in feature username admin password admin
* Add new employees
* Update employee information
* Delete employee records
* Search employees by name
* Automatic data validation
* Persistent JSON database storage
* Interactive table interface

---

## Technologies Used

* Python
* Tkinter (GUI Framework)
* JSON (data storage)
* Regular Expressions (validation)

```

```
